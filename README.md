A Flutter mobile app with authentication flow and user browsing.

## Features
- Login with JWT token authentication (user logs in once, receives a token that keeps them signed in automatically)
- Form validation (username and password required)
- Token persistence using SharedPreferences
- Auto-login if token exists
- Browse random users fetched from REST API
- Dark/Light mode toggle with persistent preference
- Bottom navigation between Home and Profile pages
- Logout clears token and redirects to welcome screen

## Tech Stack
- Flutter
- Dart
- REST API (fakestoreapi.com + randomuser.me)
- SharedPreferences

## Getting Started
1. Clone the repo
2. Run flutter pub get
3. Run flutter run
